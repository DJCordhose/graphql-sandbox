Installation
============

* Erst mal node installieren (am besten Node 6): https://nodejs.org/
* npm install

Start
=====

* npm start

API Explorer
============

* localhost:4000/graphql

JavaScript-Client

* ist in public/index.html
* starten erreichen unter: http://localhost:4000/
